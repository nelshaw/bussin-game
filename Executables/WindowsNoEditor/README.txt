WELCOME TO BUSSIN'

FEATURES 
--------
This prototype includes the following features:
*	Main menu screen
*	Pause game
*	Game over screen
*	Successful level screen
*	Collect coins
*	Large obstacles (represented as a box - ones you cannnot jump over)
*	Small obstacles (represented as a box - ones you can jump over)
*	Pick up students (represented as gold boxes)
*	Earn points by running through level (increases the longer you survive)
*	Increasing speed as gameplay continues
*	Track high score


HOW TO INSTALL
--------
1.	Download the Bussin.exe file 
2.	That's it! Enjoy!


HOW TO EXECUTE
--------
1.	Double click Bussin.exe to run the game (tested on a Windows laptop successfully)


HOW TO PLAY
--------
*	Use left and right keys to switch between lanes, and spacebar to jump
	-	If you have a touch screen, swiping left, right, and up will also control the player
*	Switch between lanes to avoid obstacles and collect coins and students
*	The top left corner indicates how many students you must pick up to complete the level

*Some resources were borrowed from vPoly DevSquad (https://www.youtube.com/c/vPoly/videos)
